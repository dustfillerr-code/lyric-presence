# assets

Put your Rich Presence image here as **`pfp.png`**.

The app's default `largeImageKey` points at the raw GitHub URL for this file:

```
https://raw.githubusercontent.com/dustfillerr-code/lyric-presence/main/assets/pfp.png
```

So once `pfp.png` is committed and pushed, Discord will show it on the presence
card automatically. To use a different image, either replace this file or set a
different URL / uploaded Discord asset key in Settings → Display → Activity image.

Recommended: square PNG, at least 512×512.
